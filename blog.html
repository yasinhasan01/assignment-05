Question-1: What are the different ways to select an element in the DOM?

Ans: Use getElementById() for a single element with a unique id. Use querySelector() for the first match using CSS selectors. Use querySelectorAll() for multiple elements with CSS selectors. Use getElementsByClassName() or getElementsByTagName() for collections but prefer querySelectorAll() for more flexibility.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Ans:innerHTML → Gets or sets HTML + text (renders HTML). innerText → Gets or sets only visible text (ignores hidden text & HTML). textContent → Gets or sets all text (including hidden text) but ignores HTML.

Question-3: What is event delegation in the DOM?

Ans:Event delegation is a technique where a parent element handles events for its child elements, even if they are added dynamically. This works using event bubbling.

Question-4: What is event bubbling in the DOM?

Ans:Event bubbling is a behavior where an event starts from the target element and propagates upwards to its ancestors.

Question-5: How do you create, add, and remove elements using JavaScript?

Ans: Use appendChild for adding, remove() for quick removal, and removeChild() for controlled deletions.
